Display category/skin in this Styleguide?
To add a new skin category:
  • Create a new Enum under /ui/ui-toolkit/ui-components/src/main/joo/com/coremedia/ui/skins/, e.g. BoxSkin.as
  • Create new documentation HTML-file in /blueprint/modules/studio/studio-styleguide-components/src/main/sencha/resources/html. The file begins with the SKIN_GROUP name as defined within the new skin enum file.
  • Add a new template in /blueprint/modules/studio/studio-styleguide-components/src/main/joo/com/coremedia/blueprint/studio/styleguide/templates
  • Add category to /blueprint/modules/studio/studio-styleguide-components/src/main/joo/com/coremedia/blueprint/studio/styleguide/tabs/skins/SkinsTreeStructure.as
  • Add title property to /blueprint/modules/studio/studio-styleguide-components/src/main/joo/com/coremedia/blueprint/studio/styleguide/Styleguide.properties
To add a new skin:
  • Simply add new Enum in the respective /ui/ui-toolkit/ui-components/src/main/joo/com/coremedia/ui/skins/*Skin.as file
  • e.g.: public static const NEW_SKIN:CategorySkin = new CategorySkin("new-skin", "This skin is awesome.");
Extend a skin category:
  • Your new skin category extends an existing skin enum. E.g. LabelableSkin.as
  • e.g.: public class NewCategorySkin extends LabelableSkin
  • The new skin enums either define custom skins
  • e.g.: public static const SPECIAL:NewCategorySkin = new NewCategorySkin('special');
  • or inherit skins from the extended skin category enum
  • e.g.: public static const DEFAULT:NewCategorySkin = new NewCategorySkin(LabelableSkin.DEFAULT.getSkin());
