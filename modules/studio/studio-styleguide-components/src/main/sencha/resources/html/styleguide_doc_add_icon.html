How to add an icon?
  1. Checkout cms-icons
  2. Add a new folder beneath src/main/icons of core-icons or collaboration-icons. The folder name is the identifier for the icon.
  3. Add the 16x16 version of the icon named using the identifier followed by <_16.svg>*
  4. (Optional) add the 24x24 version of the icon named using the identifier followed by <_24.svg>*
  5. (Optional) add the 32x32 version of the icon named using the identifier followed by <_32.svg>*
  6. Commit and push
  7. Open the cms-thirdparty-bom/pom.xml in your CMX workspace and increment the version number for cms.icons.version
  8. Open release job: https://cm7-ci.coremedia.com/job/cmx/view/releasejobs/job/release-cms-icons-1.0/, login, and trigger the build process while providing the incremented cms.icons.version number
  9. If build is successful, commit and push the adapted cms-thirdparty-bom/pom.xml

* - Although we use SVGs that can be scaled for different dimensions we decided to have specialized versions of each icon that are optimized for the different dimensions. Otherwise the icon would look blurry or important details are missing in most cases.
