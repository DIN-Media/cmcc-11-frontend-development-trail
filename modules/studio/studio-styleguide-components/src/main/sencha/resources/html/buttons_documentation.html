With button skins, you'll create awesome themes for your next project.

Take care of the scale. Not every skin is defined for all scales. The styleguide will help you. 
    <Button ui="{ButtonSkin.DEFAULT.getSkin()}"
            text="Small Example"
            iconAlign="left"
            scale="small"
            iconCls="{CoreIcons_properties.INSTANCE.star}"/>
